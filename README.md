# @boss/eslint-prettier

## Usage

### install package

```
npm i @boss/eslint-prettier -D # yarn add @boss/eslint-prettier -D
```

### use in config js

```javascript
module.exports = {
    extends: '@boss/eslint-prettier',
};
```
