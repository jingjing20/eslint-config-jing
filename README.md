# @jingjing20/eslint-prettier

## Usage

### install package

```
npm i @jingjing20/eslint-prettier -D # yarn add @jingjing20/eslint-prettier -D
```

### use in config js

```javascript
module.exports = {
    extends: '@jingjing20/eslint-prettier',
};
```
